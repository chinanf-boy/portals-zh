# 门户-portals

门户网站 是一个非常早期的提案,可以在 网站或网页 之间实现无缝导航. 特别是,该提议使页面 能够将 另一页面插入显示,并在插入状态和导航状态之间执行无缝转换. 

详细看 **[解释](https://github.com/KenjiBaheux/portals/blob/master/explainer.md)**.
